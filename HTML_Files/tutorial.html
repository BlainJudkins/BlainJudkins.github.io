MindFocus Extension Popup

Select the MindFocus logo from your extension toolbar to make the popup menu appear. Upon opening up the menu, three options will appear.

"Toggle" will allow the user to switch between displaying the bookmark and hidding it from the webpage. Toggling it on will make the bookmark appear at the top of the page.

Click the "Webstite" button to be redirected to our main website page. The "Customize" button will allow the user to customize their bookmark.

Using the Bookmark

After toggling on the bookmark, the default design will have a slit in it to narrow down the area you want uncovered.

To resize the bookmark, pull on the bottom right inner corner of the slit design. If another design is in use, pull from the bottom right corner.

To move the bookmark, you have two options: you can click and drag the bookmark around the page, or you can hold the control key and move the bookmark using the arrow keys ("ctrl" + arrows). Move the bookmark to hide any distracting text from the webpage.

Customization

If you have the Slit Design selected, use the "Slit Border Width" textbox to adjust the width of the slit border (* only works with this design).

If you would like a different design, select from the dropdown menu under "Select Design" the corresponding number of the bookmark design you wish to use.

To update the border width or bookmark design, type in your input or make your selection then click the "update" button. After doing this, open up the extension popup and re-toggle the bookmark (on-off) for your customizations to appear!
